# hotel-bouviersfriendly
Estoy en un grupo de boyeros de berna y salio la idea de montar un hotel para boyeros. Así que he usado chatGPT y dall-e para montar una web simple en html.

el prompt original fue

```
Quiero crear una web html estatica que a su vez sea responsive. 
La web sera para un hotel que ofrece alojamiento y servicios para familias con boyeros de berna. 
La estructura basica de la web constara de un header donde aparecera el nombre del hotel con enlaces a la pagina principal, tipos de habitaciones y otro a los servicios. La web tambien contará con un footer donde aparecera informacion basica sobre quienes somos, contacto y direccion física.

La web estara compuesta de las siguientes paginas:
- pagina principal: se hablará de que es un hotel en el pais de andorra, a la vall de incles, preparado para recibir perros de la raza del boyero de berna. El hotel tiene vistas a la montaña y todas las habitaciones son soleadas y con un balcon grande.
- pagina del hotel con distintos tipos de habitaciones:
	- habitacion basica: ideal parejas
	- habitación mediana: para familias
	- suite: ideal para parejas o familias que quieran un poco más de glamour.
todas las habitaciones contaran con wifi, baño privado, cama de matrimonio, cama para perro, bebedero para perros, wifi, television. Quiero que estos servicios aparezcan en un grid de 2 columnas y las filas que sean necesarias con de font-awesome.

- pagina con servicios para perros: 
	- 2 columnas con 4 filas.
	- en cada fila, abrá una columna con una foto de 960x960 y otra columna con la descripcion del servicio
	- cada fila alternara el orden anteriormente dicho. La fila uno tendra en la columna de la izquierda la foto y en la de la derecha la descripcion. En la segunda fila, la primera columna tendra la descripcion y la segunda columna la foto, y asi sucesivamente.
	- en la primera fila tendremos como servicio, masajes para perros
	- en la segunda fila tendremos distintos tipos de buffets libres
	- en la tercera fila tendremos actividades fisicas para perros, tipos piscinas, circuitos de agility o zonas de recreo
	- en la cuarta fila tendremos un servicio de guarderia para perros que tengan que quedarse solos durante unas horas



Los colores de la web deben ser con los siguientes codigos hexadecimales: 
	- color de fondo de todas las paginas: #EEE0D0
	- color de los textos: #060302
	- color de los titulos y encabezados: #DA9455

Quiero que me generes:
- la estructura de carpetas del proyecto: html, css, imagenes, js y cualquier otra que pueda necesitar.
- los distintos ficheros html con su codigo y links a las respectivas paginas
- necesito que redactes todo lo necesario para que la web resulte creible
```

Luego he ido iterando para generar los textos de los distintos servicios, pagina principal y algunos pequeños cambios de estilos.
